Patient Previous Hospitalizations

Endpoint

https://hopprlab.com/openapi/getPatientPreviousHospitalizations is the main endpoint to retrieve patients' previous hospitalizations.




Parameters

Query Parameter Value
hn Eight digit patient's hospital number.
name Can either be a patient's first name, middle name or last name.



Response

A successful request returns an Http Status Code 200 OK with the following response body :



{ "patient_id" : "00000000-0000-0000-0000-000000000000", "hospital_number" : "00000001", "last_name" : "Doe", "first_name" : "John", "middle_name" : null, "display_name" : "Doe, John", "patient_previous_hospitalizations": [ { "actual_visit_date_time" : "", "closure_date_time" : "", "visit_type_group" : "", "visit_type" : "", "primary_service" : "", "charge_type" : "" } ] }


Key Value Type Description
patient_id uniqueidentifier An id that's unique to every patient.
hospital_number string Eight digit hospital number that's unique to every patient.
last_name string Patient's last name
first_name string Patient's first name.
middle_name string Patient's middle name
display_name string Patient's full name.
actual_visit_date_time Datetime Patient's date of birth.
closure_date_time string Patient's gender.
visit_type_group string Patient's marital status.
visit_type string Patient's nationality.
primary_service string Patient's highest educational attainment.
charge_type string Patient's current occupation.
