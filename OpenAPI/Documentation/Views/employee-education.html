Employee Education Background

Endpoint

https://hopprlab.com/openapi/getEmployeeEducation is the main endpoint to retrieve employees' educational background.




Parameters

Query Parameter Value
employee_nr Four digit employee number.
name Can either be an employee's first name, middle name or last name.
old_employee_id Employee's ID number.



Response

A successful request returns an Http Status Code 200 OK with the following response body :



{ "employee_id": "00000000-0000-0000-0000-000000000000", "employee_nr": "0001", "old_employee_id": "0-00-00-0001", "display_name": "Doe, John", "last_name": "Doe", "first_name": "John", "middle_name": "null", "employee_education": [ { "major": null, "subject": null, "dates_attended_from": null, "dates_attended_to": null, "graduation_date": null, "comments": null, "verified_flag": null, "verified_date": null, "completed_flag": null, "education_type": null } ] }


Key Value Type Description
employee_id uniqueidentifier An id that's unique to every employee.
employee_nr string Four digit employee number that's unique to every employee.
old_employee_id string Employee ID.
display_name string Employee's full name.
last_name string Employee's last name.
first_name_name string Employee's first name.
middle_name string Employee's middle name.
major string Employee's major in education.
subject string The subject an employee took in his education.
dates_attended_from Datetime
dates_attended_to Datetime
graduation_date Datetime Employee's graduation date.
comments string Comments
verified_flag bool Whether employee's record is verified or not.
verified_date Datetime Date of verification.
graduation_date Datetime Employee's graduation date.
completed_flag bool Whether the employee completed the said education or not.
education_type string Can be secondary or college.
