Patient Medications

Endpoint

https://hopprlab.com/openapi/getPatientMedications is the main endpoint to retrieve patients' medications.




Parameters

Query Parameter Value
hn Eight digit patient's hospital number.
name Can either be a patient's first name, middle name or last name.



Response

A successful request returns an Http Status Code 200 OK with the following response body :



{ "patient_id" : "00000000-0000-0000-0000-000000000000", "hospital_number" : "00000001", "last_name" : "Doe", "first_name" : "John", "middle_name" : null, "display_name" : "Doe, John", "patient_medications": [ { "note_date" : "", "details" : "", "active_flag" : "" } ] }


Key Value Type Description
patient_id uniqueidentifier An id that's unique to every patient.
hospital_number string Eight digit hospital number that's unique to every patient.
last_name string Patient's last name
first_name string Patient's first name.
middle_name string Patient's middle name
display_name string Patient's full name.
note_date Datetime Patient's date of birth.
details string Patient's gender.
active_flag string Patient's marital status.
