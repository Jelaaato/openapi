Employee Career History

Endpoint

https://hopprlab.com/openapi/getEmployeeCareerHistory is the main endpoint to retrieve employees' career history.




Parameters

Query Parameter Value
employee_nr Four digit employee number.
name Can either be an employee's first name, middle name or last name.
old_employee_id Employee's ID number.



Response

A successful request returns an Http Status Code 200 OK with the following response body :



{ "employee_id": "00000000-0000-0000-0000-000000000000", "employee_nr": "0001", "old_employee_id": "0-00-00-0001", "display_name": "Doe, John", "last_name": "Doe", "first_name": "John", "middle_name": "null", "employee_career_history": [ { "career_history_start_date": null, "career_history_end_date": null, "career_history_position": null, "career_duration_in_years": null, "career_duration_in_months": null } ] }


Key Value Type Description
employee_id uniqueidentifier An id that's unique to every employee.
employee_nr string Four digit employee number that's unique to every employee.
old_employee_id string Employee ID.
display_name string Employee's full name.
last_name string Employee's last name.
first_name_name string Employee's first name.
middle_name string Employee's middle name.
career_history_start_date Datetime Employee's start date from previous career.
career_history_end_date Datetime Employee's end date from previous career.
career_history_position string Employee's position in previous career.
career_duration_in_months int Duration of employee's previous career in months.
career_duration_in_years int Duration of employee's previous career in years.
