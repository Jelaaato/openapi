Patient Diagnosis

Endpoint

https://hopprlab.com/openapi/getPatientDiagnosis is the main endpoint to retrieve patients' diagnosis.




Parameters

Query Parameter Value
hn Eight digit patient's hospital number.
name Can either be a patient's first name, middle name or last name.



Response

A successful request returns an Http Status Code 200 OK with the following response body :



{ "patient_id" : "00000000-0000-0000-0000-000000000000", "hospital_number" : "00000001", "last_name" : "Doe", "first_name" : "John", "middle_name" : null, "display_name" : "Doe, John", "patient_diagnosis": [ { "actual_visit_date_time" : "2016-01-01TT00:00:00.000", "visit_type" : "", "charge_type" : "", "coding_system" : "", "coding_type" : "", "code" : "", "diagnosis" : "", "primary_flag" : 1, "active_flag" : 1, "current_visit_diagnostic_flag" : 1, "recorded_date_time" : "2016-01-01TT00:00:00.000" } ] }
